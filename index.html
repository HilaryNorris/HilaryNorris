HTML页面跳转示例

本页面展示了多种HTML页面跳转的实现方法。您可以在下方输入URL并测试不同的跳转方式。

1. 手动跳转

输入目标URL，然后点击按钮进行跳转：

// JavaScript跳转代码
window.location.href = "目标URL";

// 新标签页打开
window.open("目标URL", "_blank");

2. 延时自动跳转

设置延时时间（秒），页面将在指定时间后自动跳转：

// 延时跳转代码
setTimeout(function() {
  window.location.href = "目标URL";
}, 延时毫秒数);

3. Meta标签自动跳转

使用HTML meta标签实现页面加载后自动跳转：

<!DOCTYPE html>
<html>
<head>
  <meta http-equiv="refresh" content="5; url=https://example.com">
</head>
<body>
  <p>5秒后将自动跳转...</p>
</body>
</html>
